## Using Your API Key

[](#using-your-api-key)

Use the API key you received as authorization for each API request you make to our sandbox. Add it to the request header as shown in this snippet from a cURL request:

-H 'Authorization: Bearer {API_KEY}'

`-H 'Authorization: Bearer {API_KEY}'`
