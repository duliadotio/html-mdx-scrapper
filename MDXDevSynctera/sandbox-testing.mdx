# Sandbox Testing

[Suggest Edits](/edit/sandbox-testing)

Triggering specific scenarios and building mock vendors to end to end test can be time consuming.  Synctera sandbox provides a few tools to help speed up testing so you can build resilient, polished apps.

[sandbox KYC test case document](/docs/sandbox-kyc-test-cases)

[simulations api reference](/reference/simulateauthorization)

Leveraging these three tools enables integrators to run end to end tests against the Synctera sandbox environment.
The following request will programmatically wipe sandbox:

`curl \
  -X POST \
  https://api-sandbox.synctera.com/v0/wipe \
  -H "Authorization: Bearer $apikey"
`

The curl example mentioned above assumes that you have set up apikey environment variables. See Authentication for instructions. Currently, this only runs in sandbox and cannot run in PROD.

`curl`

`apikey`
